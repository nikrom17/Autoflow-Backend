## Tech Stack

### Front-End

- Framework: React
- State management: Redux
- Component Library: Ant Design
- Error Tracking: Sentry, Log Rocket
- Hosting: Netlify

### Backend

- Framework: Flask
- Database: Postgres
- ORM: SQLAlchemy
- Hosting: Heroku
